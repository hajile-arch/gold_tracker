# Gold Price Tracker

A web app that scrapes gold prices from Malaysian banks and displays:

- Real-time prices (CIMB, UOB, Maybank)
- Best place to buy/sell gold
- Live dashboard (Streamlit)

## Tech Stack
- Python (Flask, BeautifulSoup)
- Streamlit
- Web Scraping

## Features
- API endpoint: /gold
- Live dashboard
- Price comparison

## Run locally
python app.py
streamlit run dashboard.py
